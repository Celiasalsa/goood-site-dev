# Goood website

## À propos

Code source du site web Goood.

* Metalsmith
* Markdown
* Browser-sync
* Sass
* Autoprefixer
* Permalinks
* Webpack


## Installation

```bash
npm install
```

## Lancer le serveur de dev

```bash
npm run dev
```

## Deployer

```bash
npm run deploy
```

Compile les fichiers et déploie sur le repo [gooodhub.github.io](https://github.com/gooodhub/gooodhub.github.io)
